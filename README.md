# Anglerfish

A personal MCP server, to enable AI supported workflows on your local machine.

## Project Targets

* Covering various tasks from writing e-Mails to visualizing data
* Easy to setup, use and configure
* Full control of my data
* Self hosted
* Open source

## Tools

- [x] Current Date and Time
- [ ] User data
- [ ] e-Mails
- [ ] Calendar
- [ ] Data visualization
- [ ] Creating diagrams
- [ ] Creating documents
- [ ] Shell scripting
- [ ] Image editing
- [ ] Weather data
- [ ] Routplanning and traffic data
