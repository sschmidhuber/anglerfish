# Anglerfish

A personal MCP server, to enable AI supported workflows on your local machine.

## Project Targets

* Covering various tasks from writing e-Mails to visualizing data
* Easy to setup, use and configure
* Full control of my data
* Self hosted
* Open source

# Ideas

## Tools

- [x] current date and time
- [x] access user data
- [x] compose e-Mails
- [x] create calendar items
- [/] file system access (read directory, search files, move, copy, delete, rename, read file, read file as, write file, write specific files)
- [ ] create documents
- [ ] Shell scripting
- [ ] Data visualization
- [ ] Creating diagrams
- [ ] Image editing
- [ ] Weather data
- [ ] Routplanning, traffic and geo data

## Installation Options

- [ ] Dependency management
